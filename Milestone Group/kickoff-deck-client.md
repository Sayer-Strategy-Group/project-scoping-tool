# Key Risks

### Data Quality Risk (Medium severity / High likelihood)
**Impact:** Contacts scattered across Excel, OneNote, and email may contain duplicates and inconsistent formatting, increasing migration effort
**Mitigation:** Conduct data audit before import; deduplicate in staging area; small test loads before full migration; set up HubSpot duplicate detection

### Adoption Risk (High severity / Medium likelihood)
**Impact:** Team members with varying tech comfort (ages 25–65) may resist CRM adoption, reverting to Excel/OneNote habits
**Mitigation:** Start with minimal required fields; automate capture; role-specific training; leadership champions usage; phased rollout builds confidence

### Scope Creep (Medium severity / Medium likelihood)
**Impact:** Additional feature requests before Phase 1 is stable could delay timeline
**Mitigation:** Enforce phased approach; document future features as out of scope for current phase; change order process for additions

### Stakeholder Availability (Medium severity / Medium likelihood)
**Impact:** Key decision-makers distributed across 4 offices with active deal flow, making scheduling difficult
**Mitigation:** Recurring weekly check-ins; async communication for approvals; one primary POC (Jason Wise) for decisions
