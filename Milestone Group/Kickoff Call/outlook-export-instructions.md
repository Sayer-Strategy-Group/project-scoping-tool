# How to Export Your Contacts from Outlook

*We need each of you (Jason, Robert, David, Rich, Jim) to export your contacts separately. This gives us the starting data for HubSpot. There are two types of exports -- do both if you can.*

---

## Export 1: Your Formal Contacts (Address Book)

These are contacts you've explicitly saved in Outlook.

### Outlook Desktop (Windows)

1. Open Outlook desktop app
2. Click **File** > **Open & Export** > **Import/Export**
3. Select **Export to a file** > click Next
4. Select **Comma Separated Values (.csv)** > click Next
5. Select the **Contacts** folder (make sure it's your main Contacts, not a subfolder) > click Next
6. Choose a save location and name the file (e.g., `jason-contacts-export.csv`) > click Finish
7. Send the CSV to Kyle

### Outlook on the Web (outlook.office.com)

1. Go to outlook.office.com and sign in
2. Click the **People** icon (bottom-left, person silhouette) -- or navigate to outlook.office.com/people
3. Click **Manage contacts** (top toolbar) > **Export contacts**
4. Under "Which contacts do you want to export?" select **All contacts**
5. Click **Export** -- it will download a CSV file
6. Send the CSV to Kyle

### Outlook for Mac

1. Open Outlook > click **People** in the bottom navigation
2. Select all contacts (Cmd+A) or the specific folder you want
3. Click **File** > **Export**
4. Note: Mac Outlook may not have a direct CSV export. If you don't see it, use the Outlook on the Web method above instead.

## Export 2: Your Email Activity Contacts (The Hidden Gold)

These are people you've **emailed with** but never formally saved as contacts. Outlook tracks these automatically and they're often the most valuable data source -- especially for broker relationships where you've been corresponding but never added them to your address book.

### Outlook Desktop (Windows)

1. Open Outlook desktop app
2. Click **File** > **Open & Export** > **Import/Export**
3. Select **Export to a file** > click Next
4. Select **Comma Separated Values (.csv)** > click Next
5. In the folder list, look for **Suggested Contacts** or navigate to the **Recipient Cache** / **Auto-Complete list**
   - If you see "Suggested Contacts" as a folder, select it and export
   - If not, the auto-complete cache is stored differently -- we can grab it on a call
6. Name the file (e.g., `jason-suggested-contacts.csv`) > click Finish

### If You Can't Find "Suggested Contacts"

Don't worry -- this one can be tricky to locate depending on your Outlook version. We can do this together on a quick 15-minute call. Just do Export 1 above and we'll handle this one live.

## Export 3: Zoho CRM Contacts (If Applicable)

Some of the team has been using Zoho for specific purposes. If you have contacts or accounts in Zoho that should come over to HubSpot:

1. Log into Zoho CRM
2. Go to **Contacts** (or **Leads** / **Accounts** depending on what you've been using)
3. Click the **three-dot menu** (top-right of the list view) > **Export**
4. Select **Export Contacts** > choose **CSV format**
5. Export all records -- we'll filter on our end
6. Repeat for any other modules you've been using (Accounts, Deals, etc.)

If the Zoho export gives you trouble or the data is in a weird format, just let us know. We have tools to clean up and transform whatever you send.
