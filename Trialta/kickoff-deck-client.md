# Key Risks

### Data Quality (Medium severity / High likelihood)
**Impact:** Contact data across multiple personal spreadsheets may contain duplicates and inconsistent formatting, increasing migration effort
**Mitigation:** Structured import template provided in Week 1; data audit before committing to migration plan; small test loads before full import

### Adoption (Medium severity / Medium likelihood)
**Impact:** Team accustomed to personal networks and memory-based relationship management may resist structured CRM usage
**Mitigation:** Start with Outlook integration for immediate value; minimize required fields; automate capture; make CRM the easiest path, not extra work

### Data Submission Timing (Medium severity / Medium likelihood)
**Impact:** Team must provide raw data files by end of Week 2 or migration timeline compresses
**Mitigation:** Import template sent during kickoff; clear Week 2 deadline; if delayed, we pull forward pipeline work

### Scope Creep (Low severity / Medium likelihood)
**Impact:** Desire to add GTM engine work (demand gen, collateral, positioning) into this engagement
**Mitigation:** This engagement covers CRM infrastructure only; future GTM phases documented separately; change order process for additions
