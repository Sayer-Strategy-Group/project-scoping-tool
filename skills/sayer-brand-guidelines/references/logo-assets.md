## SVG Fallbacks (HTML/React only)

For HTML or React artifacts that need the logo inline, ask the operator for the SVG
assets or fetch the file from Drive by ID and embed its contents. For all other
formats (docx, pptx, xlsx), fetch from Drive by file ID instead.
